# Keyboard-Only Users

Not everyone can use a mouse or trackpad. Some people navigate entirely with a keyboard — Tab to move between elements, Enter or Space to activate them, arrow keys to move within components, and Escape to dismiss.

**Who:** People with motor impairments (cerebral palsy, muscular dystrophy, repetitive strain injuries, paralysis), power users who prefer the keyboard, and all screen reader users (since screen readers are keyboard-driven).

**What they use:**
- **Standard keyboard** — Tab, Shift+Tab, Enter, Space, arrow keys, Escape
- **Switch devices** — one or two buttons that cycle through focusable elements (used by people with severe motor impairments)
- **Sip-and-puff devices** — controlled by breath, mapped to keyboard actions
- **Head-tracking and eye-tracking systems** — translate movement into cursor or keyboard input

**What they rely on:** Every interactive element reachable by keyboard. Visible focus indicators showing where they are. A logical tab order that matches the visual layout. No keyboard traps — if you can Tab into something, you must be able to Tab out. Alternatives to drag-and-drop and hover interactions.

**Quick check:** *Can I complete the entire task with Tab, Enter, and arrow keys?*
